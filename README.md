# control-on-screen

<h2>Requirements</h2>  

<p>python: v3.11</p>  
<p>numpy</p>  
<p>mediapipe</p>  
<p>pyautogui</p>  
<p>open-cv</p>  



<h2>installation</h2>
after install Python 3.11 you can run these commands:

<code>pip install opencv-python</code>
<code>pip install numpy</code>
<code>pip install mediapipe</code>
<code>pip install pyautogui</code>
